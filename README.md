# Snowflake ハンズオン

Tasty Bytes（架空のフードトラック会社）のデータを使って、Snowflakeの機能を体験してみましょう！

## このハンズオンで学べること

### 1. データパイプライン (`notebook/1_Simple_Data_Pipeline.ipynb`)
- 半構造化データ（JSON）の扱い方
- Dynamic Tableで自動更新
- Taskでスケジューリング

### 2. データコラボレーション (`notebook/2_Data_Share.ipynb`)
- データシェアリングで他の組織とデータ共有
- MarketPlaceからデータを取得

### 3. BI＆AI (`streamlit/3_streamlit_dashboard.py`)
- Streamlitでダッシュボード作成
- Snowflake Intelligenceでデータ分析

## やってみよう

1. `setup.sql`を実行して環境をセットアップ
2. ハンズオン1でデータパイプラインを作ってみる
3. ハンズオン2でデータ共有を試してみる
4. ハンズオン3でダッシュボードを作ってみる
